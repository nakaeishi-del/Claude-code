# TomoMeet (友会)

友達グループの集まりを自動スケジューリングするWebアプリ。

## セットアップ

```bash
npm install
npx prisma db push
npm run db:seed
npm run dev
```

## 機能

1. 友達グループの自動スケジューラー
   - グループ作成・メンバー招待
   - 空き時間の設定
   - AIによる最適な日程・レストランの提案
   - 賛否投票システム

2. イベント発見カレンダー（Coming Soon - 2026年秋予定）
