# ウェビナーナビ

ヨンデミー保護者向けウェビナーの運用を、知識がなくても画面の指示に従うだけで完結させるナビゲーションアプリです。

## 起動方法

```bash
# 依存パッケージのインストール（初回のみ）
npm install

# 開発サーバー起動
npm run dev
# → http://localhost:5173 でアクセス可能

# 本番ビルド
npm run build
npm run preview  # ビルド結果の確認
```

## 主な機能

| 機能 | 説明 |
|------|------|
| ダッシュボード | 今日のタスク、進捗状況、全体サマリー |
| ウェビナー管理 | 開催予定の一覧・詳細・削除 |
| タスクガイド | 13種類のタスクを手順書形式でナビゲート |
| スマートペースト | テキストを貼り付けるだけでフォーム自動入力 |
| 文面テンプレート | 7種類のメール・SNS文面を変数自動置換してコピー |
| 履歴・実績 | 操作ログとウェビナー別進捗の可視化 |

## ファイル構成

```
src/
├── types/         # 型定義
├── data/          # タスクテンプレート・文面テンプレート・サンプルデータ
├── utils/         # localStorage操作・タスク生成
├── components/    # 共通UIコンポーネント
└── pages/         # 各画面
```

## データ保存

すべてのデータはブラウザの **localStorage** に保存されます。
サーバー不要・ログイン不要でそのまま使えます。

設定画面からJSONエクスポートでバックアップが可能です。
